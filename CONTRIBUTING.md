# 🤝🏻 CONTRIBUTING
## 👬 How to Contribute
**You can help the project by:**
- Reporting bugs and issues.
- Submitting suggestions.
- Helping to improve the project's code.
## 🧑🏻‍💻 Reporting Issues
**Please include the following when reporting an issue:**
- A description of the issue
- Steps to reproduce the issue
- Expected result
- Actual result
## ⚖ Guidelines
**You are also required to follow these rules:**
- Code must be readable and organized.
- Major changes must be coordinated with the core development team before implementation.
- Project guidelines must be followed.
