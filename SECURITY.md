# 🔒 SECURITY Policy
## Report security issues
#### ‼️If you find a security issue, please do not publish it publicly in Issues!
### 📧 To report a security issue, contact us via this GMAIL: HPC.Studio.SUPPORT@gmail.com
## Information Required for the Report
Please include the following details when submitting a report:
- Description of the issue
- Steps to reproduce the issue
- Version being used
- Python version being used
- Any other information necessary to investigate the issue
## ⏰ Response Time
1️⃣ Security reports will be reviewed as soon as possible.
## 📲 Supported Versions
✅ Currently only the latest released version is supported...!
